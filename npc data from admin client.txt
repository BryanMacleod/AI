Pathway editor commands:
load zone's paths: /path load
save to database: /path save
list all paths: /path list
create a path: /path create <path name>
delete a path: /path delete <path name>
draw a path: /path draw <path name>|all
have an npc run a path: /path npc <npc name> <path name>
move a path to the ground: /path ground <path name>
list nodes in path: /path node list <path name>
add a node: /path node add <path name> <order, defaults to last>
move a node: /path node move <path name> <order>
delete a node: /path node delete <path name> <order>

Usage: /reportheat local - reports all heat on the admin's zone
/reportheat allworlds [HeatCategoryName] [MinimumHeatLevelToReport] - queries all worlds for the heat matching the constraints.  Empty constraints will dump *all heat* on all worlds.
Example: /reportheat allworlds Movement.ServerSpeedCheck 10
/reportheat player <playername> - will dump all heat records for the specified player

defaults are [heat per event] = 0.2, [heat decay] = 1.0, and [heat limit] = 2.5

Usage of /npc: 
/npc health <hp> [name/guid] -Sets the npcs hp to specified value, Admin Target Compatible.
/npc admintarget guid
/npc disableSpawners 0 | 1
/npc spawn <spawnId>
/npc despawn GUID | all | vehicles | allexcept GUID | everythingbyowner GUID
/npc damage damageAmount NpcName
/npc create NpcDefinitionId SpawnDelaySeconds RespawnDelaySeconds
/npc create2 NpcDefinitionId {Quantity} {Radius}
/npc find {partialstringmatch} - find the defs that have this string in their name or designer name
/npc BypassRelevance <0/1>/n/npc info [guid] -compatible with Admin Target
/npc loc NpcName
/npc locs NpcDefinitionID -list the locations of npc's by npcdefid on the server/npc setowner [npcGuid] newOwnerGuid (npcGuid will be taken from admin target if not included)
